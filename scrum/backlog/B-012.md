## Descricao

O comando `mdgraph tree` percorre o grafo inteiro sem limite de profundidade.
Em repositorios grandes isso produz saidas extensas e dificeis de ler.

Adicionar flag `--depth INT` para limitar quantos niveis sao exibidos.

## Sintaxe

```bash
# Apenas o no raiz (sem filhos)
mdgraph tree arquivo.md#id --depth 0

# Apenas filhos diretos
mdgraph tree arquivo.md#id --depth 1

# Dois niveis de profundidade
mdgraph tree arquivo.md#id --depth 2

# Sem limite (comportamento atual, padrao)
mdgraph tree arquivo.md#id
```

## Criterios de aceite

- [ ] `--depth 0` exibe apenas o no raiz (sem filhos)
- [ ] `--depth 1` exibe filhos diretos (sem netos)
- [ ] Sem `--depth` o comportamento e identico ao atual (profundidade ilimitada)
- [ ] Funciona tambem com `--refs` (backlinks)
- [ ] Testes cobrindo depth 0, 1 e ilimitado
