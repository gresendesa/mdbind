## Description

Add instructions for installing `mdbind` globally as a CLI tool using `pipx` to the user-facing documentation. This provides a cleaner alternative to installing with global/user `pip` without virtual environment clutter.

## Scope

- Document installation via `pipx install mdbind` (from PyPI) and `pipx install git+https://github.com/gresendesa/mdbind.git` (from source) in `README.md`.
- Explain prerequisites (Python, pipx).
- Explain how to run `mdb` commands after pipx installation.

## Out of scope

- Packaging mdbind into OS-specific packages (e.g. apt, brew, pacman).
